# Projeto Docker e Microsserviços

Projeto acadêmico sobre Docker e Microsserviços, desenvolvido através de 5 desafios progressivos.

## Objetivos

- Dominar conceitos de containerização com Docker
- Implementar comunicação entre microsserviços
- Aplicar boas práticas de desenvolvimento
- Desenvolver soluções escaláveis

## Desafios

### Desafio 1: Containers em Rede (20 pontos)

**Status**: Completo

Comunicação entre containers via rede customizada.

- Tecnologias: Docker Networks, Python, Flask
- [Ver Desafio 1](./desafio1-containers-rede/)

### Desafio 2: Docker Compose (20 pontos)

**Status**: Em desenvolvimento

- Tecnologias: Docker Compose, PostgreSQL, React

### Desafio 3: API RESTful (20 pontos)

**Status**: Planejado

- Tecnologias: FastAPI, Redis, MongoDB

### Desafio 4: Monitoramento (20 pontos)

**Status**: Planejado

- Tecnologias: Prometheus, Grafana, ELK

### Desafio 5: CI/CD (20 pontos)

**Status**: Planejado

- Tecnologias: GitHub Actions, Docker Hub
